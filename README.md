  What I Created

  All source code in this repository was written by me to fulfil the assignment requirements. This includes:

  Program.cs - Main Entry Point

  - An interactive console menu system that lets the user navigate through all 23 topics across three sections

  Section A: Software Testing Concepts (SectionA_SoftwareTestingConcepts.cs)

  Working C# demonstrations of:
  - Functional Testing (discount calculator with pass/fail assertions)
  - Non-Functional Testing (performance/response time simulation)
  - Load Testing and Stress Testing simulations
  - Black-Box Testing and White-Box Testing examples
  - Equivalence Partitioning and Boundary Value Analysis
  - Error-Case Testing with exception handling
  - Test Case Structure definitions
  - Test Results Interpretation logic
  - Test Summary Report generation

  Section B: Basic C# Programming (SectionB_ConsoleAndForms.cs)

  - Console output, input handling, and string interpolation
  - Variables, data types, and operators
  - Selection structures (if/else, switch)
  - Loop structures (for, while, foreach)
  - Arrays and Lists with CRUD operations
  - Methods and functions
  - Two Windows Forms applications: a Greeting Form and a Calculator Form
  - A menu-driven Student Console App with add, display, search, and delete functionality

  Section C: Practical Application (SectionC_PracticalApplication.cs)

  - A complete Student Record Management System built as a Windows Forms application, featuring:
    - Student data model with properties and validation
    - Add, Delete, Find, and Clear functionality
    - ComboBox for course selection
    - ListBox display of all records
    - Input validation and error handling
    - Proper control naming conventions (btn, txt, lbl, cmb, lst, grp)
  - 17 detailed test case definitions (12 functional, 5 non-functional)
  - A defect report with 4 defects at varying severity levels and fix recommendations
  - A professional software testing report with scope, results summary, defect list, recommendations, and conclusion
  - A GitHub workflow guide covering git commands and branching strategy
  - A submission evidence checklist
